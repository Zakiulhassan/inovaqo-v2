Abstract brand shapes with psychological meanings — for section markers, value walls, deck dividers, social tiles.

```jsx
<ShapeGlyph kind="control" size={72} />              // controlled velocity
<ShapeGlyph kind="openRing" color="#EAF2EF" />        // openness / no black boxes
<ShapeGlyph kind="continuity" color="var(--accent)" /> // creative momentum
```

Pair each shape with its meaning in a mono label (e.g. `SHAPE / CONTROLLED VELOCITY`). Use 1–2 per composition, large and confident — never as scatter decoration. At most one lime accent element per shape.
