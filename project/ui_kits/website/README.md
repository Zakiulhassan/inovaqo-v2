# Website UI kit

Self-contained recreation of the INOVAQO marketing homepage per Module 7's IA:
Hero → Trust strip → Problem → INOVAQO System (services) → Build Clarity Framework → Selected Work → Why INOVAQO → Final CTA.

Built with plain HTML/CSS against the token system (`../../styles.css`) so it renders without the component bundle. When building production pages, use the React primitives in `components/` instead of copying these inline styles.

Rules embodied here:
- Hero leads with "AI product engineering for bold ideas that need to scale." + real proof (75+/98%/2-wk) — never placeholder zeros.
- Services render as one connected system (linked modules), not 5 generic boxes.
- Exactly one lime CTA per viewport; signal accents are rare.
- Case studies are large editorial blocks: Challenge / System / Outcome + tags; image area is a placeholder for real product shots.
